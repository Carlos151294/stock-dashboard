# IPC Stock Market Dashboard

Stock Dashboard is a project which seeks to reflect the [IPC](https://www.eleconomista.es/diccionario-de-economia/indice-de-precios-y-cotizaciones-ipc) from MÃ©xico.

## Installation

Install initial dependencies:

    npm install

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm test`

Launches the test runner in the interactive watch mode.\

### `npm deploy`

Deploys the app to [https://carlos151294.github.io/stock-dashboard](https://carlos151294.github.io/stock-dashboard) which makes use of [Github Pages](https://pages.github.com/).\
